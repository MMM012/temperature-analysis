# Анализ температурных данных

Приложение для анализа исторических данных о температуре и мониторинга текущей погоды через OpenWeatherMap API.

## Функционал

- Анализ исторических температурных данных
- Вычисление скользящего среднего (30 дней)
- Определение аномалий (температура за пределами mean ± 2σ)
- Статистика по сезонам
- Получение текущей температуры через OpenWeatherMap API
- Проверка аномальности текущей температуры

## Файлы

- `analysis.py` — функции анализа данных и распараллеливание
- `weather_api.py` — работа с API (синхронный и асинхронный методы)
- `app.py` — Streamlit приложение
- `temperature_data.csv` — исторические данные
